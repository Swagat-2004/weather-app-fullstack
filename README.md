# weather-app-fullstack
Full Stack Weather Application built using React (Frontend) and Spring Boot (Backend).  The app fetches real-time weather data using a weather API and displays temperature, humidity, and weather conditions.

# 🌦 Weather Finder

Full Stack Weather Application built with:

- React (Frontend)
- Spring Boot (Backend)
- OpenWeather API
- Vercel (Frontend Deployment)
- Railway (Backend Deployment)
